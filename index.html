Easy Kext Installer

Description

EasyKext Pro offers you the possibility to deploy custom kernel extensions on your system without dealing with complicated procedures. The utility comes with drag-and-drop support and an auto-mode that starts the installation right away.

Features

Unsophisticated installer tool that can deal with kernel extensions

The EasyKext Pro application comes with a minimalist graphical user interface that helps you deal with the kexts selection process and configure the installation. However, the utility does not integrate a collection of predefined custom extensions: you will have to provide the files yourself. EasyKext Pro detects and displays the operating system's version in the bottom area of its main window, so make sure to use compatible packages. Note that you can load kext, bundle, plugin, or PPP files, either by using the Finder menu or by dropping them on top of the app's window in the appropriate area. The app will keep a list of the loaded files, which you get to clear with the press of a button.

Change the installation location and backup your kernel extensions

The EasyKext Pro main panel also offers you the possibility to specify the installation location for the extension packages: either the Library/ Extensions or the System/Library/Extensions directories. At the same time, you can configure the EasyKext Pro utility to backup your kexts, in which case the files will be sent to the desktop. Keep in mind that if the auto-mode is activated, EasyKext Pro will start the installation process as soon as you load the files, so make all the necessary adjustments beforehand.

Deploy kernel extensions to bypass hardware limitations on non-Apple machines

Bottom line, EasyKext Pro is able to install custom kexts, plug-ins, and bundles on your system with minimal effort. This proves to be especially useful if you are working on a Hackintosh machine and you need to deploy custom drivers to bypass hardware limitations.
